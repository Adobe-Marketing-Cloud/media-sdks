----------------------------------
Adobe Mobile Library (Chromecast)
----------------------------------
Current Version:2.2.0

=======
SUMMARY
=======
ADBMobile Library for Chromecast lets you measure Chromecast applications written in JS, leverage and collect audience data through audience management, and measure video engagement through Video Heartbeats.

================
PACKAGE CONTENTS
================
<package-dir>/Readme.txt
<package-dir>/ReleaseNotes.txt
<package-dir>/libs/adbmobile-chromecast.min.js               - ADBMobile Library for Chromecast
<package-dir>/docs/                                          - API Docs 
<package-dir>/samples/BasicPlayerSample/*                    - Sample Chromecast Receiver Application

========================
SETTING UP SAMPLE PLAYER
========================
Chromecast receiver sample is included in the release package.

To set up the receiver sample. 
Download and copy the following version of player.js (https://raw.githubusercontent.com/googlecast/CastReferencePlayer/ebfe3f46e771e3fbcd3be10759ef2563d9bbc54b/player.js)
into "<package-dir>/samples/BasicPlayerSample/script/google" directory.
