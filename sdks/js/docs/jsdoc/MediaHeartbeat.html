trackEvent(event, info, data)

Method to track an event in media playback.

Parameters:
Name Type Description
event MediaHeartbeat.Event

Event to be tracked.
info object

MediaObject of the Event to be tracked.
MediaHeartbeat.Event#AdBreakStart should have MediaObject created using MediaHeartbeat.createAdBreakObject
MediaHeartbeat.Event#AdStart should have MediaObject created using MediaHeartbeat.createAdObject
MediaHeartbeat.Event#ChapterStart should have MediaObject created using MediaHeartbeat.createChapterObject
data object

Context data for MediaHeartbeat.Event#AdStart and MediaHeartbeat.Event#ChapterStart events. Pass null for other events.
