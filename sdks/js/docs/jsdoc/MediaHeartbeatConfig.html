MediaHeartbeatConfig

new MediaHeartbeatConfig()

Creates an instance of MediaHeartbeatConfig class

Members

trackingServer

hearbeat tracking server property. Required.

channel

channel name property, defaults to empty string.

ovp

Name of the online video platform through which content gets distributed. Default value is "unknown".

appVersion

Version of the video player app/SDK. Default value is "unknown".

playerName

Name of the video player on use. i.e. "AVPlayer", "HTML5 Player", "My Custom VideoPlayer".

ssl

Property that indicates whether the heartbeat calls should be made over HTTPS. Default value is false.

debugLogging

debugLogging property, must be set true to enable debug logging for Heartbeat.
